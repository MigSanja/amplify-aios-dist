# Outreach-Messages: Vorlage (generisch)
> Das ist die **Nachrichten-Vorlage** des Outreach-Agenten. Fülle sie im Onboarding (`/aios-setup`) mit deiner echten Stimme, deinem Angebot und echten Referenzen.
> Regeln: **Du-Form**, KEINE Signatur, KEINE Gedankenstriche, keine Garantie in der DM. Ein Gerüst, pro Zielgruppe eine Variante (CRM-Feld `category`).

## Icebreaker (vor M1, pro Lead dynamisch)
Genau EIN Profil-Anker (bevorzugt ein aktueller Post) · KEIN Pitch/Produkt · **Du-Form: „Hey [Vorname],"** + Leerzeile + [kurze eigene Einordnung] · kein CTA · keine Gedankenstriche · max. 200 Zeichen. Quelle: CRM-Feld `icebreaker`.

## M1: Erstkontakt (Icebreaker + Leerzeile + Variante nach `category`)
```
<HIER_DEIN_PAIN_AUFHÄNGER: 2-3 Sätze, die das Problem deiner Zielgruppe in ihren Worten beschreiben.>

Kurz zu mir: Ich bin <DEIN_NAME> und <WAS_DU_BAUST/ANBIETEST> für <ZIELGRUPPE>, ohne dass sie dafür jemanden einstellen müssen. <EIN_KONKRETER_REFERENZ_CASE mit echter Zahl, sobald du einen hast: bis dahin weglassen, KEINE erfundene Zahl.>

<DEIN_ANGEBOT/HAKEN: z.B. gemeinsam den größten Hebel finden und einen ersten Schritt kostenlos liefern.>

Hättest du Lust, dass wir da mal kurz sprechen?
```
> Lege bei Bedarf weitere Varianten je `category` an (z.B. M1-BRANCHE-A / M1-BRANCHE-B). Gleiches Gerüst, angepasster Pain + Referenz.

## M2: Follow-up 1 (nur Text)
```
Hey [Vorname],

kurze Nachfrage: Ist das Thema <DEIN_THEMA> bei euch grundsätzlich interessant?

Falls ja, <DEIN_LOW-RISK-ANGEBOT>.
```

## M3: Follow-up 2 (optional, Bild)
Ein lockeres Bild/Meme + kurzer Text. Bilder liegen in `$HOME/AIOS/assets/`.

> Regel: Max. 2-3 weiche Follow-ups pro Lead, dann Schluss. InMail-Leads bekommen KEINE Follow-ups.
